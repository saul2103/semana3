# Comparación entre Programación Tradicional y Programación Orientada a Objetos

## Autor
Bryan Saul Iza Llano

## Descripción
Este proyecto contiene dos paradigmas de programación utilizados para obtener soluciones e información básica.

## Temas
- Programación Tradicional.
- Programación Orientada a Objetos.

## Reflexión
La programación tradicional se enfoca en los procesos y nos dicta paso a paso para resolver un problema, mientras que la Programación Orientada a Objetos se centra en los datos y organiza el código en componentes independientes que imitan el mundo real. Esto hace que el estilo tradicional sea ideal para tareas sencillas y directas, pero vuelve al enfoque orientado a objetos mucho más útil para mantener, escalar y reutilizar código en proyectos grandes, ya que cada objeto cuida sus propios datos sin afectar al resto del sistema.
